# Done Yet? demo renderer

This Remotion project burns the review captions onto the real public-console capture. The imported source video is intentionally gitignored; copy `demo/video/done-yet-base.mp4` to `demo/video-edit/public/done-yet-base.mp4` before rendering.

```bash
npm install
npm run lint
npm run render
```

The output is `demo/video/done-yet-build-week-captioned.mp4`.
